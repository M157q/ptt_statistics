# ptt_statistics  
  
Get the statistics of a board on PTT.  
  
---  
  
## Installation  
  
`$ pip install git+https://github.com/M157q/ptt_statistics`  
  
---  
  
## Usage  
  
```  
usage: ptt-statistics [-h] board  
  
positional arguments:  
  board  
  
optional arguments:  
  -h, --help  show this help message and exit  
```
